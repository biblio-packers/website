マイフェス2019Webサイト サーバー移行による一部機能停止のお知らせ

マイフェス2019イベントWebサイト(https://myfes.biblio-packers.net)は、サーバーの安定化・サイトの軽量化を目的に、Amazon Web Serviceが提供する「AWS Lightsail」へのサーバーの移行を実施することとなりました。実施に際し、データベースの整合性を取るためにWebサイトで利用可能な一部の機能の提供を一時停止致します。

実施期間: 2019年8月27日～8月29日(予定)


利用できなくなる機能: マイフェス2019 e-ショップでの販売商品閲覧、購入など全ての取引


ご迷惑をおかけ致しますが、何卒ご了承、ご協力をよろしくお願い致します。
